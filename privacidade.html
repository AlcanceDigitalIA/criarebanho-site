Cria Rebanho Pro

Política de Privacidade

Última atualização: 17 de junho de 2026

Resumo: O Cria Rebanho Pro armazena seus dados principalmente no seu próprio celular. Coletamos apenas o mínimo necessário para o funcionamento do app e nunca vendemos seus dados.

1. Quem somos

O Cria Rebanho Pro é um aplicativo de gestão de rebanho bovino desenvolvido por Anderson Lopes de Oliveira, disponível para dispositivos Android. Para dúvidas, entre em contato pelo e-mail contato@criarebanho.app.br.

2. Quais dados coletamos

O aplicativo pode coletar e armazenar os seguintes dados:

  • Dados de cadastro: endereço de e-mail e senha, utilizados para criar sua conta e permitir o login.
  • Dados da fazenda: nome da propriedade, município, estado e telefone de contato — preenchidos pelo próprio usuário nas configurações.
  • Dados do rebanho: informações sobre animais, eventos sanitários, pesagens, reprodução, piquetes e lançamentos financeiros — todos inseridos pelo usuário.
  • Dados de uso do plano: identificação do plano (gratuito ou premium) e validade da assinatura.

3. Como os dados são armazenados

Os dados são armazenados de duas formas:

  • Localmente no dispositivo: todos os dados são salvos no próprio celular do usuário usando banco de dados SQLite. O app funciona completamente offline.
  • Na nuvem (Supabase): quando há conexão com a internet, os dados são sincronizados com nosso servidor na nuvem para backup e acesso em múltiplos dispositivos. Os dados são transmitidos com criptografia (HTTPS).

4. Para que usamos os dados

  • Permitir o funcionamento do aplicativo e o acesso à sua conta
  • Sincronizar seus dados entre dispositivos
  • Fazer backup seguro das suas informações na nuvem
  • Verificar o status do plano Premium
  • Enviar alertas e notificações de eventos cadastrados pelo usuário

5. Compartilhamento de dados

Não vendemos, alugamos nem compartilhamos seus dados pessoais com terceiros para fins comerciais. Os dados podem ser compartilhados apenas com:

  • Supabase: provedor de banco de dados na nuvem, utilizado exclusivamente para armazenamento e sincronização dos dados do app.

6. Notificações

O aplicativo pode enviar notificações locais para lembretes de eventos sanitários, partos previstos e piquetes a vencer, com base nos dados que você mesmo cadastrou. Você pode desativar as notificações a qualquer momento nas configurações do seu celular.

7. Seus direitos

Você tem o direito de:

  • Acessar todos os dados que cadastrou no aplicativo
  • Corrigir ou atualizar seus dados a qualquer momento
  • Excluir sua conta e todos os dados associados — basta entrar em contato pelo e-mail abaixo
  • Exportar seus dados em formato PDF através dos relatórios do app

8. Segurança

Adotamos medidas técnicas para proteger seus dados, incluindo transmissão criptografada (HTTPS/TLS) e autenticação segura. Recomendamos que você utilize uma senha forte e não compartilhe suas credenciais de acesso.

9. Crianças

O Cria Rebanho Pro não é destinado a menores de 18 anos e não coleta intencionalmente dados de crianças ou adolescentes.

10. Alterações nesta política

Podemos atualizar esta Política de Privacidade periodicamente. Quando houver alterações relevantes, notificaremos os usuários pelo próprio aplicativo. O uso contínuo do app após as alterações implica na aceitação da nova política.
